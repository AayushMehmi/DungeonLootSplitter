Logic Explanation

My loot array is the source of truth because all loot items are stored there as objects. Each object includes a name, value, and quantity, so the app can calculate value times quantity. The updateUI() function recalculates totals, renders the loot list, and updates what is visible or disabled. Conditional logic prevents invalid loot, invalid party size, and empty state errors from becoming part of the app state.
